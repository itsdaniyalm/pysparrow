Installation

The most easiest method for installation of pieSparrow is through PyPi and pip

Use pip to install pieSparrow in your Python environment
$pip install piesparrow

as an alternate, you can download the complete package from GitHub

Initialization

Before composing anything, the html file needs to be initialize through init function.

Function:
init(filename, title, icon)
Parameter Format Description
filename string name of HMTL file which will be generated
title string Title of HMTL page
icon True/False Turn on or off the pieSparrow banding on top right of page
Example:
import piesparrow as ps
 ps.init(filename='helloWorld.html', title='Hello World !', icon=True)

Grid System

Grids are composed of rows and column, each row can contain a minimum of 1 and maximum of 5 columns. Unlimited number of rows can be added

Row and Columns

Row is a basic building component, each visual element that needs to be presented on the page must be contained with in a row and atleast one column. row is called as a function while columns are its parameters

Function:
row(col1, col2, col3, col4, col5, col1w, col2w, col3w, col4w, col5w)
Parameter Format Description
col1 string/variable/function Contents that need to be displayed in first column
col2 string/variable/function Contents that need to be displayed in second column
col3 string/variable/function Contents that need to be displayed in third column
col4 string/variable/function Contents that need to be displayed in fourth column
col5 string/variable/function Contents that need to be displayed in fifth column
col1w integer Width of first column, can be defined between 10 to 100 with increments of 10 like 10,20,30,...,100
col2w integer Width of second column, can be defined between 10 to 100 with increments of 10 like 10,20,30,...,100
col3w integer Width of third column, can be defined between 10 to 100 with increments of 10 like 10,20,30,...,100
col4w integer Width of fourth column, can be defined between 10 to 100 with increments of 10 like 10,20,30,...,100
col5w integer Width of fifth column, can be defined between 10 to 100 with increments of 10 like 10,20,30,...,100
Example:
ps.row(
    col1w = 50,
    col1 = ('This is the First Column of 50% width'),
    col2w = 50,
    col2 = ('This is the Second Column of 50% width')
)
Output:
This is the First Column of 50% width
This is the Second Column of 50% width

Typography

Beside writing text without any function will result in text being displayed, four different heading sizes starting from h1 (biggest) to h4 (smallest) can be called. p can be called for paragraph texts and bold can be called for strong text.

Function:
h1(text)
h2(text)
h3(text)
h4(text)
p(text)
bold(text)
Parameter Format Description
text string/variable/function Contents that needs to be displayed
Example:
ps.row(
    col1 = (
        ps.h1('This is H1 Text')+
        ps.h2('This is H2 Text')+
        ps.h3('This is H3 Text')+
        ps.h4('This is H4 Text')+
        ps.p('This is paragraph Text')+
        ps.bold('This is bold text')+
        ps.p('This is '+ps.bold('inline bold')+'text')
    )
)
Output:

This is H1 Text

This is H2 Text

This is H3 Text

This is H4 Text

This is paragraph text

This is bold text

This is inline bold text

Images

Images can be displayed by calling img.

Function:
img(path, height, width)
Parameter Format Description
path string Path of image along with filename and extension
height integer Height of image, default is 250
width integer Width of image, default is 250
Example:
ps.row(
    col1 = ps.img(path='images/logo.png', height=100, width=360)

Dataframes

Pandas data frames can be beautifully displaye by callingtable.

Function:
table(df, justify)
Parameter Format Description
df dataframe Pandas dataframe object that needs to be displayed
justify string Text alignment of column headers, left, center of right
Example:
data = pandas.read_csv('population.csv')
ps.row(
    col1 = ps.table(df=data, justify='left')

Bar Charts

Bar Charts can be created by callingbar function.

Function:
bar(title, dataframe, xlabel, ydata, ylabel, titledisplay, legenddisplay , color, height, grid, axis)
Parameter Format Description
title string Title of Chart
dataframe Pandas dataframe object Dataframe for which chart needs to be displayed
xlabel string Name of column in dataframe which needs to be displayed on x-axis
ydata string Name of column in dataframe for values which needs to be plotted on y-axis
ylabel string Label to be displayed on tool tip
titledisplay string 'true' displays the chart title, 'false' turn off the chart title
legenddisplay string 'true' displays the legends, 'false' turn off the legends
color list Colors hex codes can be passed as a list
or one of the pieSparrow color pallet can be used
height integer Height of chart (width is controlled by column width)
grid string 'true' displays the grid, 'false' turn off the grid
axis string 'true' displays the axis, 'false' turn off the axis
Example:
df = pandas.read_csv('population.csv')
ps.row(
    col1 = ps.bar(
        title='Population of Pakistani Cities',
        dataframe = df,
        xlabel = 'City',
        ydata = 'Population 2020',
        ylabel = 'Population in 2020',
        legenddisplay = 'false',
        height= 50,
        color = ps.rainbow
    )
)

Line Charts

Line Charts can be created by callingline function. A maximum of five lines can be plotted in single chart.

Function:
line(title, dataframe, xlabel, ylabel1, ydata1, ylabel2, ydata2, ylabel3, ydata3, ylabel4, ydata4,
ylabel5, ydata5, titledisplay, legenddisplay, color, height, grid, axis)
Parameter Format Description
title string Title of Chart
dataframe Pandas dataframe object Dataframe for which chart needs to be displayed
xlabel string Name of column in dataframe which needs to be displayed on x-axis
ylabel1 string Label which needs to be displayed for first line
ydata1 string Name of column in dataframe for values which needs to be plotted on y-axis as first line
ylabel2 string Label which needs to be displayed for second line
ydata2 string Name of column in dataframe for values which needs to be plotted on y-axis as second line
ylabel3 string Label which needs to be displayed for third line
ydata3 string Name of column in dataframe for values which needs to be plotted on y-axis as third line
ylabel4 string Label which needs to be displayed for fourth line
ydata4 string Name of column in dataframe for values which needs to be plotted on y-axis as fourth line
ylabel5 string Label which needs to be displayed for fifth line
ydata5 string Name of column in dataframe for values which needs to be plotted on y-axis as fifth line
titledisplay string 'true' displays the chart title, 'false' turn off the chart title
legenddisplay string 'true' displays the legends, 'false' turn off the legends
color list Colors hex codes can be passed as a list
or one of the pieSparrow color pallet can be used
height integer Height of chart (width is controlled by column width)
grid string 'true' displays the grid, 'false' turn off the grid
axis string 'true' displays the axis, 'false' turn off the axis

df = pandas.read_csv('population.csv')
ps.row(

        ylabel3 = 'Punjab',
        ydata3 = 'Punjab',

        legenddisplay = 'true'

Pie Charts

Pie Charts can be created by callingpie function.

Function:
pie(title, dataframe, labels, data, titledisplay, legenddisplay, color, height, width , grid, axis)
Parameter Format Description
title string Title of Chart
dataframe Pandas dataframe object Dataframe for which chart needs to be displayed
labels string Name of column in dataframe which needs to be displayed as lables
data string Name of column in dataframe for values which needs to be plotted in pie chart
titledisplay string 'true' displays the chart title, 'false' turn off the chart title
legenddisplay string 'true' displays the legends, 'false' turn off the legends
color list Colors hex codes can be passed as a list
or one of the pieSparrow color pallet can be used
height integer Height of chart (width is controlled by column width)
grid string 'true' displays the grid, 'false' turn off the grid
axis string 'true' displays the axis, 'false' turn off the axis
Example:
df = pandas.read_csv('population.csv')
pps.row(
    col1 = ps.pie(
        title = 'Population of Cities',
        dataframe = df,
        labels = 'City',
        data = 'Population 2020',
        height = 400,
        color = ps.dusktilldawn
    )
)

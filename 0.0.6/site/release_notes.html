Version 0.1.2 - Beta

Release Data: December 13, 2022 | PyPi

New Features

- This release is a complete re-write of base library.

- The base charting library have now been changed to Billboard.JS

- New custom made CSS framework sparrow.css have been implemented.

- Color themes have been reenvisioned, these can now be applied in init function by using parameters basetheme and charttheme .

- New column types added, now column can be defined as colxs (extra small) colsm ( small), colmd (medium), collg (large), colxs (extra large).

- Columns can now have shadows which can be imlementd by usign type parameter while defining columns.

- Line, Spline, Area and Bar charts have now been combined into one function and be called by chart.

- donut, gauge and kpi chart types has been added.

- Ability to render links has been added which can be called using link function.

- The parameters of all functions have been changed.

Discontinued

- Dependency on milligram.css has been deprecated.

- The chart type of line and bar from version 0.0.5 has been deprecated

- The ability of applying color theme for every chart has been discontinued.

- The convention of defining column as col1, col2, col3, col4, col5 has ben discontinued.
